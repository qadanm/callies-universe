Sticker-like multi-select pill for the optional context step (lifestyle / vibe / relationship-to-car). Selected chips fill ember and show a ✕ so they're easy to deselect.

```jsx
<Chip selected={picked} onToggle={setPicked}>Weekend warrior</Chip>
<Chip emoji="🏁">Track days</Chip>
```

Keep groups fast to multi-select; chips are skippable seasoning, never required.
