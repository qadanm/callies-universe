Glossy inflatable button — the main action affordance. Squishes on press and springs back; display-font label.

```jsx
<Button variant="primary" size="lg" block>Roast my car 🔥</Button>
<Button variant="secondary">Maybe later</Button>
<Button variant="ghost">Skip</Button>
```

Variants: `primary` (dominant ember CTA), `accent` (bright flame, ink label), `secondary` (outline on white), `ghost` (text). Sizes `sm | md | lg` (lg = the hero CTA height). Supports `iconLeft` / `iconRight`, `block`, `disabled`.
