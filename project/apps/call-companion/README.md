# Call Companion — app skin (Callie's Universe)

The **second app** in Callie's Universe, here as proof the shared core powers more than one product. It inherits the *entire* core (tokens, components, Callie, the cast) and overrides only its **accent** (a calm periwinkle/indigo) plus its **hero screen**.

- `theme.css` — the only color override: `--accent-*` set to indigo. Nothing else.
- `index.html` — the hero: an **incoming-call screen** where a cast character (Abu Omar) calls and **Callie reacts** (she answers-delighted, comfort on decline). Answer → on-call state.

Same Callie art, same Roaster avatars, same Button/tokens as RoastMyRide — only the accent + this screen differ. Change Callie's `savage` art or a core token and both apps inherit it; apps never fork the core.
